# Mini Task Dashboard

A modern full-stack task management application built with Next.js and Supabase that helps users create, manage, update, and track tasks efficiently.

# live demo: https://task-master-tan-tau.vercel.app/

## Features

- Create tasks
- View all tasks
- Update/Edit tasks
- Delete tasks
- Task status management
  - Todo
  - In Progress
  - Completed
- Due date support
- Responsive UI
- Custom toast notifications

## Setup Instructions

### Clone repository

```bash
git clone https://github.com/GauRaVsinghbora/taskMaster.git
```

### Move into project

```bash
cd mini-task-dashboard
```

### Install dependencies

```bash
npm install
```

### Create environment file

Create a `.env.local` file and add:

```env
NEXT_PUBLIC_SUPABASE_URL=your_supabase_url
NEXT_PUBLIC_SUPABASE_ANON_KEY=your_supabase_key
```

### Run project

```bash
npm run dev
```

Open:

```txt
http://localhost:3000
```
